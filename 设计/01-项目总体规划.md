# 家庭幼儿诗词故事平台 — 项目总体规划

> 文档版本：v2.0  
> 编写日期：2026-06-04  
> 覆盖范围：唐诗宋词朗读 + 儿童故事 + 后台资料/音视频管理

---

## 一、项目概述

### 1.1 产品定位

面向**家庭自用**的幼儿文化启蒙平台，核心能力为：

1. **唐诗宋词朗读**：展示诗词原文，播放预录或 TTS 朗读，支持跟读
2. **讲故事**：分段展示儿童故事，播放音频或视频
3. **后台管理**：家长通过管理端维护诗词/故事文本，上传并绑定音视频资源

平台不面向公众运营，无需多租户、班级、考试、错题等 K12 学习功能。

### 1.2 目标用户

| 角色 | 说明 |
|------|------|
| 家长（管理员） | 后台登录，维护内容、上传音视频、发布/下架 |
| 幼儿（听众） | 家庭端浏览诗词与故事，听读观看（需家长陪同使用） |

### 1.3 设计原则

- **内容优先**：文本 + 优质音视频，而非刷题与分数
- **够用即可**：单体后端、单管理员，避免微服务过度设计
- **离线友好**：支持本地/NAS 部署，数据不出家庭网络
- **屏幕可控**：单次播放建议 3–8 分钟，支持定时停止

## 二、功能范围

### 2.1 家庭端（前台）

| 编号 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| F-01 | 首页推荐 | P0 | 今日 1 首诗 + 1 个故事 |
| F-02 | 诗词列表 | P0 | 按朝代、难度筛选，仅展示已发布 |
| F-03 | 诗词朗读页 | P0 | 大字展示、拼音开关、音频播放、逐句高亮 |
| F-04 | 故事列表 | P0 | 按类型（童话/寓言/成语）筛选 |
| F-05 | 故事播放页 | P0 | 分段正文、音频/视频播放、段落高亮 |
| F-06 | TTS 降级 | P1 | 无预录音频时使用浏览器 Web Speech API |
| F-07 | 收藏 | P2 | localStorage 收藏诗词/故事 |
| F-08 | 播放记录 | P2 | 记录最近播放、完成状态 |
| F-09 | 定时停止 | P2 | 哄睡模式，N 分钟后自动暂停 |

### 2.2 管理端（后台）

| 编号 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| A-01 | 管理员登录 | P0 | 单账号 JWT 认证 |
| A-02 | 诗词 CRUD | P0 | 增删改查、草稿/发布、排序 |
| A-03 | 故事 CRUD | P0 | 分段编辑、类型、适龄、发布 |
| A-04 | 媒体库 | P0 | 上传音频/视频/封面，列表、预览、删除 |
| A-05 | 媒体绑定 | P0 | 诗词/故事绑定封面、朗读音频、讲解视频 |
| A-06 | 引用校验 | P1 | 删除媒体前检查是否被内容引用 |
| A-07 | 批量导入 | P2 | JSON/Excel 批量导入诗词文本 |
| A-08 | **文字转音频** | **P0** | Edge TTS 合成 MP3，绑定诗词/故事 |

### 2.3 明确不做

- 用户注册、多角色权限体系
- 题库、错题本、考试、知识图谱
- 学习报告、AI 解题、班级管理
- 微服务、API 网关、消息队列（首期）

### 5.2 表结构

#### media_asset（媒体资源）

```sql
CREATE TABLE media_asset (
    id            BIGSERIAL PRIMARY KEY,
    file_name     VARCHAR(255) NOT NULL,
    file_path     VARCHAR(500) NOT NULL,
    media_type    VARCHAR(20) NOT NULL,    -- AUDIO / VIDEO / IMAGE
    mime_type     VARCHAR(100),
    file_size     BIGINT,
    duration_sec  INT,
    created_at    TIMESTAMP DEFAULT NOW()
);
```

#### poem（诗词）

```sql
CREATE TABLE poem (
    id            BIGSERIAL PRIMARY KEY,
    title         VARCHAR(100) NOT NULL,
    author        VARCHAR(50),
    dynasty       VARCHAR(20),             -- 唐 / 宋
    lines         JSONB NOT NULL,            -- ["床前明月光", ...]
    pinyin        JSONB,                     -- 与 lines 一一对应，可选
    translation   TEXT,                      -- 白话释义
    difficulty    VARCHAR(20) DEFAULT 'easy', -- easy / medium
    tags          JSONB,                     -- ["名篇","思乡"]
    cover_id      BIGINT REFERENCES media_asset(id),
    audio_id      BIGINT REFERENCES media_asset(id),
    video_id      BIGINT REFERENCES media_asset(id),
    status        VARCHAR(20) DEFAULT 'draft', -- draft / published
    sort_order    INT DEFAULT 0,
    created_at    TIMESTAMP DEFAULT NOW(),
    updated_at    TIMESTAMP DEFAULT NOW()
);
```

#### story（故事）

```sql
CREATE TABLE story (
    id            BIGSERIAL PRIMARY KEY,
    title         VARCHAR(200) NOT NULL,
    story_type    VARCHAR(30),               -- 童话 / 寓言 / 成语 / 原创
    paragraphs    JSONB NOT NULL,            -- 分段正文
    age_min       INT DEFAULT 3,
    cover_id      BIGINT REFERENCES media_asset(id),
    audio_id      BIGINT REFERENCES media_asset(id),
    video_id      BIGINT REFERENCES media_asset(id),
    status        VARCHAR(20) DEFAULT 'draft',
    sort_order    INT DEFAULT 0,
    created_at    TIMESTAMP DEFAULT NOW(),
    updated_at    TIMESTAMP DEFAULT NOW()
);
```

#### admin_user（管理员）

```sql
CREATE TABLE admin_user (
    id            BIGSERIAL PRIMARY KEY,
    username      VARCHAR(50) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    created_at    TIMESTAMP DEFAULT NOW()
);
```

## 六、API 设计

### 6.1 规范

- 基础路径：`/api/v1`
- 管理端认证：`Authorization: Bearer <jwt_token>`
- 统一响应：

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

### 6.2 公开接口（家庭端）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/home/recommend` | 今日推荐（1 诗 + 1 故事） |
| GET | `/api/v1/poems` | 已发布诗词列表（dynasty、difficulty 筛选） |
| GET | `/api/v1/poems/{id}` | 诗词详情（含媒体 URL） |
| GET | `/api/v1/stories` | 已发布故事列表（storyType 筛选） |
| GET | `/api/v1/stories/{id}` | 故事详情 |
| GET | `/api/v1/media/stream/{id}` | 音视频流（或 302 至预签名 URL） |

### 6.3 管理接口

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/v1/admin/auth/login` | 登录，返回 JWT |
| POST | `/api/v1/admin/auth/logout` | 登出（可选） |
| GET | `/api/v1/admin/poems` | 诗词列表（含 draft） |
| POST | `/api/v1/admin/poems` | 新增诗词 |

## 八、播放逻辑

### 8.1 音频优先级

```
1. 后台绑定的预录 audioUrl → HTML5 <audio> 播放
2. 无预录 → Web Speech API (speechSynthesis) 朗读
3. 有 videoUrl → 可选视频模式，音频与视频二选一或音画同步
```

### 8.2 逐句/逐段高亮

- **诗词**：按 `lines` 数组长度拆分；TTS 模式按句依次 speak；预录模式可用字幕时间轴（P2，需在媒体 metadata 存 `lineTimestamps`）
- **故事**：按 `paragraphs` 分段；播放进度与段落索引映射

### 8.3 useSpeech Hook（TTS 降级）

```typescript
// hooks/useSpeech.ts — 概念接口
function useSpeech() {
  return {
    speak: (text: string, options?: { rate?: number }) => void,
    pause: () => void,
    resume: () => void,
    cancel: () => void,
    isSpeaking: boolean,
  };
}
```

## 九、内容规划（首期种子数据）

### 9.1 诗词（建议 15–20 首）

**唐诗（easy）**：静夜思、悯农、咏鹅、春晓、登鹳雀楼、望庐山瀑布、江雪、枫桥夜泊、游子吟、草

**宋词（medium，可选）**：西江月·夜行黄沙道中、清平乐·村居

### 9.2 故事（建议 10–15 篇）

- 童话：三只小猪、龟兔赛跑、小红帽
- 寓言：守株待兔、刻舟求剑、狐假虎威
- 成语：画龙点睛、亡羊补牢、滥竽充数

每篇 3–8 段，预估 3–5 分钟。

## 十一、安全与配置

### 11.1 管理端安全

- 密码 BCrypt 存储
- JWT 过期时间建议 24h（家庭场景可延长）
- 管理 API 统一前缀 `/api/v1/admin/**`，Spring Security 拦截
- 上传文件校验 MIME 与扩展名白名单

### 11.2 配置示例（application.yml 片段）

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/family_edu?useUnicode=true&characterEncoding=utf8&serverTimezone=Asia/Shanghai
    username: root
    password: root

app:
  storage:
    local-path: ./uploads
  upload:
    max-audio-size: 20971520    # 20MB
    max-video-size: 209715200   # 200MB
    max-image-size: 5242880     # 5MB
  jwt:
    secret: ${JWT_SECRET}
    expire-hours: 24
  tts:
    enabled: true
    provider: edge
    voice: zh-CN-XiaoxiaoNeural   # 中文女声
    rate: +0%
    timeout-sec: 60
```

## 十二、实施计划

| 阶段 | 内容 | 产出 |
|------|------|------|
| **P0** | 后端：实体、媒体上传、诗词/故事 CRUD、公开只读 API | 可 Postman 联调 |
| **P1** | 管理端：登录 + 诗词/故事/媒体库页面 | 可录入并发布内容 |
| **P2** | 家庭端：列表、详情、音频播放、TTS 降级 | 可听读 |
| **P3** | 逐句高亮、收藏、播放记录、定时停止 | 体验完善 |
| **P4** | Docker 部署、MinIO、批量导入、TTS 生成 | 运维与效率 |

## 十三、验收标准

- [ ] 管理端可登录，新增诗词并上传 MP3，发布后家庭端可见
- [ ] 管理端可新增故事（分段），绑定封面与音频
- [ ] 家庭端诗词页可播放预录音频；无音频时 TTS 可读
- [ ] 家庭端故事页可分段展示并播放
- [ ] 媒体库删除被引用文件时有明确提示
- [ ] 本地部署下上传文件重启后仍可访问
